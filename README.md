mpu6050_cli
===========

MPU6050 IC cli tools


This code provide basic communication with MPU6050 Six-Axis (Gyro + Accelerometer) MEMS MotionTracking(tm) Devices direcly from cli using I2C.
Currently tested on BeagleBone Black.
