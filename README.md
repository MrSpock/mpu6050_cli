mpu6050_cli
===========

MPU6050 IC cli tools


This tool (will be) is for accessing MPU6050 
Six-Axis (Gyro + Accelerometer) MEMS MotionTracking(tm) Devices direcly from cli using I2C.
Currently tested on BeagleBone Black.
